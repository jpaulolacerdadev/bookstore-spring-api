# 📚 Bookstore API

API REST em desenvolvimento para gerenciamento de uma livraria, construída com Java e Spring Boot.  
O projeto tem como objetivo aplicar conceitos de desenvolvimento backend, criação de APIs REST e persistência de dados utilizando banco de dados relacional.

---

## 🚀 Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

---

## ⚙️ Funcionalidades

- Cadastro de livros
- Listagem de livros
- Atualização de registros
- Remoção de livros
- Integração com banco de dados PostgreSQL

---

## 🧱 Arquitetura

O projeto segue boas práticas de desenvolvimento backend utilizando:

- Arquitetura em camadas (Controller, Service, Repository)
- API REST
- Persistência de dados com JPA/Hibernate

---

## 🧪 Testes de API

Os endpoints podem ser testados utilizando ferramentas como:

- Postman
- Insomnia

---

## 🚧 Status do Projeto

Projeto em desenvolvimento, com novas funcionalidades e melhorias sendo adicionadas continuamente como parte do processo de aprendizado e prática em desenvolvimento backend.

---

## 📌 Objetivo

Praticar o desenvolvimento de aplicações backend utilizando o ecossistema Spring, reforçando conceitos de orientação a objetos, integração com banco de dados e construção de APIs REST.
